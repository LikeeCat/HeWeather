
# HeWeather
##项目简介
一个接近完成版的天气预报
##项目要求
请使用Xcode 7.3 打开项目
本项目使用的数据库是Realm 请使用Realm Browser Version 0.103.2 (76)查看数据库文件

##一些提示
这个项目有一些依赖的第三方库,如果你下载好了项目无法正常的运行,请手动Pod install一下
